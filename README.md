## General info
this project is an simple shopping cart

## Features
* add product to cart
* prevent adding same product to cart


## Technologies
Project is created with:
* Vue 2.6.14
* vuetify ^2.6.1
* vue-router ^3.5.3
* vuex ^3.6.2
* axios ^0.24.0

## Status
completed
	
## Setup
To run this project, install it locally using npm:

```
$ cd Shopping-cart-with-vue-2-
$ npm install
$ npm run serve
```
